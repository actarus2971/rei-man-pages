### **0.2 Installazione**

REI è un’applicazione **Java multipiattaforma**.

Il pacchetto include:

- runtime Java integrato
- gestione automatica della compatibilità

📌 In caso di mismatch:

- REI aggiorna automaticamente la JVM

---

Avvio installazione per Sistema Operativo

|**Sistema**|**Metodo**|
|---|---|
|Windows|`./run_REI.bat`|
|Linux|`./run_REI.sh` (richiede `chmod +x`)|
|macOS (.app)|doppio clic su REI.app|
|macOS (.dmg)|trascinare in Applications|
|macOS (script)|`./run_REI.sh`|

📌 macOS:

- al primo avvio può richiedere autorizzazione manuale per la sicurezza
- i log sono salvati in `~/.caravel-rei.log`
