### **0.1 Download del Software**

REI è distribuito come parte dell’ecosistema **CARAVEL**.

Il software è disponibile tramite portale dedicato:

- URL: http://caravel.int.ingv.it
- Sezione: _REI – Desktop application for seismic data review and analysis_
- Versione: scaricare la **Latest Release** (es. v3.39.0)

![../../attachments/Caravel Main Page.png\|408](/img/user/Manuale%20di%20REI/attachments/Caravel%20Main%20Page.png)
📌 È raccomandato utilizzare l’ultima versione per garantire compatibilità con i servizi:

- Dante
- Apollo
